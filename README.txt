== Description ==

**Install plugin to disable woocommerce product from your admin product page**

Easy to use, you only install the plugin.

*We test our plugins with many other. If you find incompatibility with one specific plugin, please contact me on <a href="http://purvik.me">purvik.me</a>.*

== Installation ==

1. Upload `woo-product-disable.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
